# 🩸 HemoConnect

# Smart Blood Donation & Emergency Healthcare Platform

HemoConnect is a futuristic blood donation and emergency healthcare management platform developed using PHP, HTML, CSS, JavaScript, Bootstrap, and MySQL.

This project helps users find blood donors, raise emergency blood requests, use AI-powered healthcare tools, manage donation appointments, and spread awareness about blood donation through a premium modern UI.

---

# ⚙️ Installation

## 1️⃣ Install XAMPP

Download and install XAMPP or WAMP.

---

## 2️⃣ Start Apache & MySQL

Open XAMPP Control Panel and start:

- Apache
- MySQL

---

## 3️⃣ Download Project

Download ZIP or clone repository.

---

## 4️⃣ Move Project Folder

Move the project folder into:

```text
C:\xampp\htdocs\
```

---

## 5️⃣ Open phpMyAdmin

Open browser and visit:

```text
http://localhost/phpmyadmin
```

---

## 6️⃣ Create Database

Create a new database named:

```text
blood_bank_database
```

---

## 7️⃣ Import SQL File

Import the SQL database file into phpMyAdmin.

---

## 8️⃣ Run Project

Open browser and visit:

```text
http://localhost/HemoConnect/home.php
```
